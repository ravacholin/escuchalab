# EscuchaLAB - AI-Powered Spanish Listening Practice

EscuchaLAB is a sophisticated web application designed to help users practice their Spanish listening skills through AI-generated content. The application leverages Google's Gemini API to create dynamic and level-appropriate dialogues, audio, and exercises. It also integrates with Freesound for ambient audio to create a more immersive experience.

## Getting Started

To run this project locally, you will need to have Node.js installed on your machine.

1.  **Install Dependencies:**

    ```bash
    npm install
    ```

2.  **Set Up Environment Variables:**

    Create a `.env.local` file in the root of the project and add your Gemini API key:

    ```
    GEMINI_API_KEY=your_key_here
    ```

3.  **Run the Application:**

    ```bash
    npm run dev
    ```

    The application will be available at `http://localhost:3000`.


## Folder Structure

The project is organized into the following directories:

-   `components/`: Contains all the React components used in the application.
-   `data/`: Includes the data sources, such as the scenarios for the exercises.
-   `public/`: Stores the public assets, such as images and logos.
-   `services/`: Contains the services that interact with external APIs, like the Gemini API.

## File Descriptions

Below is a detailed description of each file in the project:

### Root Files

-   **`App.tsx`**: The main component of the application. It manages the application's state and renders the different screens.
-   **`CLAUDE.md`**: A file with instructions for the Claude Code AI.
-   **`index.html`**: The main HTML file of the application.
-   **`index.tsx`**: The entry point of the React application.
-   **`package.json`**: The project's dependencies and scripts.
-   **`tsconfig.json`**: The TypeScript configuration file.
-   **`types.ts`**: The TypeScript types and enums used in the application.
-   **`vite.config.ts`**: The Vite configuration file.

### Components

-   **`AudioPlayer.tsx`**: A component that plays the audio generated by the Gemini API. It also handles the ambient audio from Freesound.
-   **`AuthScreen.tsx`**: A component that prompts the user to enter their Gemini API key.
-   **`ExerciseCard.tsx`**: A component that renders the exercises for the user to complete.
-   **`LoadingScreen.tsx`**: A component that is displayed while the application is generating the lesson plan and audio.
-   **`MatrixSelector.tsx`**: A component that allows the user to select the scenario for the exercise.
-   **`SelectInput.tsx`**: A generic component for a select input.

### Data

-   **`scenarios.ts`**: Contains the scenarios for the exercises, organized by level.

### Public

-   **`escucha-logo-cropped.png`**: The cropped logo of the application.
-   **`escucha-logo.png`**: The full logo of the application.
-   **`escucha-logo.svg`**: The SVG version of the logo.

### Services

-   **`geminiService.ts`**: A service that interacts with the Gemini API to generate the lesson plan and audio.
